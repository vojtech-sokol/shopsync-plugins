# getInventoryDocumentFile

**Category:** Documents
**HTTP:** `POST https://api.baselinker.com/connector.php`
**Body:** `method=getInventoryDocumentFile&parameters=<json>`
**Source:** <https://api.baselinker.com/index.php?method=getInventoryDocumentFile>

## Description

The method allows you to download a warehouse document PDF file.

## Input Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| document_id | int | Yes | Warehouse document identifier from BaseLinker. |
| get_external | bool | No | Default `false`. When `false`, retrieves document generated by BaseLinker; when `true`, attempts to download from external accounting system. Falls back to BaseLinker format if external file unavailable. |

## Response

| Field | Type | Description |
|-------|------|-------------|
| status | varchar(30) | `SUCCESS` or `ERROR`. On error, returns additional `error_message` and `error_code` fields. |
| file | text | Document PDF file in binary format encoded in base64, with prefix `data:`. |
| document_number | varchar(100) | Document number. |

## Example request

```json
{
  "document_id": 4521
}
```

## Example response

```json
{
  "status": "SUCCESS",
  "file": "data:4AAQSkZJRgABA[...]",
  "document_number": "WZ/2026/03/001"
}
```
